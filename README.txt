1. Checkout this repo
2. Change the images as you required ( optionally change the names as well )
3. Database properties are mentioned in DBUtil.java to load from the external properties file called "conf/gaadidost-db.properties". 
4. Configure your database configuration properties.
5. Build a war
6. Run the SQL files. Here, we given Schema & tables creation in one file and sample data in anonther sql file.
7. After that, you can deploy the war.
